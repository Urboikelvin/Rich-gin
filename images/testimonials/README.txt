This folder already contains illustrated avatar graphics for the three
sample testimonials (client1.svg, client2.svg, client3.svg) — you don't
need to add anything for the site to look complete.

To use a REAL client photo instead of an illustration:
1. Save the photo as client1.jpg (or .png) in this folder — square photos
   around 200x200px work best since they display as circles.
2. Open index.html, find the matching testimonial, and change its
   src="images/testimonials/client1.svg" to
   src="images/testimonials/client1.jpg"

To add a 4th (or more) testimonial, copy one whole .tslide block in the
TESTIMONIALS section of index.html, add a matching .tdot button, and give
the new avatar its own image file the same way.
